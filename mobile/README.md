# Taallam Mobile App

## تشغيل على موبايلك مباشرة

### الخطوة 1: تثبيت الـ dependencies
```bash
cd mobile
npm install
```

### الخطوة 2: تشغيل الـ app
```bash
npx expo start
```

### الخطوة 3: فتح على الموبايل
- حمّل **Expo Go** من App Store أو Google Play
- Android: scan الـ QR code
- iOS: scan الـ QR code بالكاميرا

## Features
- ✅ Screen Capture Prevention (iOS كامل، Android جزئي)  
- ✅ Supabase Auth
- ✅ Course Enrollment System
- ✅ Lesson Player (Vimeo)
- ✅ Challenges with Gemini AI
